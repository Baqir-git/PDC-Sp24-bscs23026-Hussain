This repository contains the solution for PDC Assignment 4. The assignment analyzes three distributed systems failures in a FastAPI + React application (StudySync) and implements a fix for one of them.

|---|---|---|

## Key Implementation Details

### Custom Middleware Header (Required)

Every API response includes the header `X-Student-ID: bscs23026`, enforced by `StudentIDMiddleware` registered on the FastAPI app.

```python
class StudentIDMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request, call_next):
        response = await call_next(request)
        response.headers["X-Student-ID"] = "bscs23026"
        return response
```

### Circuit Breaker States

```
          3 failures
CLOSED  ────────────►  OPEN  ──── 10s timeout ────►  HALF_OPEN
  ▲                                                       │
  └───────────────── probe succeeds ─────────────────────┘
```

- **CLOSED** — normal operation, requests pass through
- **OPEN** — LLM is down, instant fallback returned (no waiting)
- **HALF_OPEN** — one probe allowed to test if LLM recovered
